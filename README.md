# Python Practice Tasks

A collection of 7 Python programming tasks for practicing core programming concepts, data structures, and problem-solving.

---

## Tasks Overview

### 1. Contact Book Manager (task-01.py)

**Description**: Manage contacts with CRUD operations

- Add, view, search, and delete contacts
- Dictionary operations
- Input validation

---

### 2. Student Grades Manager (task-02.py)

**Description**: Manage student grades and calculate statistics

- Store student grades
- Calculate average, min, max
- Data processing and calculations

---

### 3. Expense Tracker (task-03.py)

**Description**: Track personal expenses

- Add expenses with categories
- Calculate totals and summaries
- File I/O operations
- Data aggregation

---

### 4. URL Parser (task-04.py)

**Description**: Parse and extract components from URLs

- Extract protocol, domain, path, parameters
- String parsing with regex
- Data validation

---

### 5. Task Scheduler (task-05.py)

**Description**: Manage tasks with priority and deadlines

- Add tasks with priority (1-5) and deadlines
- Sort by priority and deadline
- OOP implementation
- Custom sorting logic

**Features**:

- Priority-based sorting
- Deadline management
- Task validation

---

### 6. Item Inventory (task-06.py)

**Description**: Manage product inventory with constraints

- Add/remove items
- Track quantity
- Prevent overselling
- Low stock warnings

**Features**:

- CRUD operations
- Stock threshold alerts
- Inventory validation

---

### 7. Log File Analyzer (task-07.py)

**Description**: Parse and analyze log files

- Count log entries by level (INFO, ERROR, WARNING)
- Find busiest hour
- Identify most common error messages
- File parsing and statistics

**Input Format**: `[YYYY-MM-DD HH:MM] LEVEL Message`

---

## How to Run

Each task is a standalone Python script:

```bash
# Run individual tasks
python task-01.py
python task-02.py
python task-03.py
# ... etc
```

---

## Learning Objectives

- **Data Structures**: Lists, dictionaries, sets
- **File I/O**: Reading and writing files
- **String Manipulation**: Parsing, formatting, regex
- **OOP**: Classes, methods, encapsulation
- **Algorithms**: Sorting, searching, filtering
- **Error Handling**: Input validation, exceptions

---

## Requirements

- Python 3.7+
- Standard library only (no external dependencies)
