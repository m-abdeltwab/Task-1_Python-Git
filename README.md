

## 1. Smart Number Analyzer

User enters a list of numbers. Program prints:

- Minimum value
- Maximum value
- Average value
- How many numbers are prime
- How many numbers are even
- How many numbers are odd

--------------------------------------------------

## 2. Text Statistics Engine

User enters a paragraph. Program outputs:

- Total characters
- Total words
- Unique words
- Most common word (ignore case and punctuation)

--------------------------------------------------

## 3. Password Strength Validator

Validate a password using the following rules:

- Minimum length
- At least:
  - 1 uppercase letter
  - 1 lowercase letter
  - 1 digit
  - 1 special character
- No repeated sequence longer than 2 characters
